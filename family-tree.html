    Adding someone new

    The whole tree is built from one list in this file's code, called treeData. Every person is an object that can have its own children list — so the tree can grow to any size or depth without redesigning anything.

    { name: "Kabir Kundu", years: "b. 2005", role: "Son", children: [] }

    To add a child to someone, find their entry in treeData near the bottom of this file, and add a new object like the one above inside their children: [ ] list. To add a whole new couple to an existing generation, copy an existing sibling's whole block and edit the details. Save the file, upload it back to GitHub, and the tree updates automatically.
